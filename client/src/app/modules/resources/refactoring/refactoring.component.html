Code refactoring
  • Refactor or even redesign part of your app if there's enough benefit. A few examples:
    • Tree-traversing recursive function converts dollar strings to numbers for any tree-like data structure (trading client-side CPU cycles for code base reduction).
    • Promise.all() enables Operating System multi-threading while saving data in a loop (order does not matter).
    • sequelize models + JavaScript functions reduce back end code by nearly 80% compared to using raw SQL queries.
